# Proyecto Lampreas Violeta 

Este proyecto ha sido desarrollado para el módulo de Acceso a datos (2º DAM). Consiste en una aplicación Java diseñada para gestionar la persistencia y manipulación de datos de una empresa.

## Características
- Patrón DAO (Data Access Object): Implementación de interfaces para la separación de la lógica de negocio y el acceso a datos (ClienteDAO, PedidoDAO, ProductoDAO, etc.).

- Gestión de JSON: Capacidad para importar y exportar datos en formato JSON mediante la clase JsonIO.

- Arquitectura por capas: Organización del código en paquetes (model, dao, services, app) para facilitar el mantenimiento.

## Tecnologías utilizadas
- Java: Lenguaje principal de desarrollo.

- Maven: Gestión de dependencias y construcción del proyecto (archivo pom.xml).

- JSON: Formato de intercambio de datos.
